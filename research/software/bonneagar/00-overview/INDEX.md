# Web Development Research Index

This directory contains research on web frameworks, authentication, and frontend technologies.

## Framework Directories

### Full-Stack Frameworks
- **tanstack/** - TanStack Start full-stack React framework
  - Architecture, routing, SSR patterns
  - Integration with other tools (Effect, Better Auth)

- **hono/** - Edge-first web framework
  - Full-stack dashboard patterns
  - Integration with Drizzle, oRPC

### Backend & Real-Time
- **convex/** - Real-time backend-as-a-service
  - Core features, authentication
  - Effect integration research

### Authentication
- **betterauth/** - Modern authentication framework
  - Setup guides, self-hosting patterns
  - Integration with Convex, Tanstack Start

### Type-Safe APIs
- **orpc/** - Type-safe RPC framework
  - OpenAPI generation, contract-first design

### UI Frameworks
- **gradio/** - Python-based UI framework
  - OpenAPI research, integration patterns

### Functional TypeScript
- **effect/** - Effect-TS functional programming
  - TanStack Start integration
  - Convex integration
