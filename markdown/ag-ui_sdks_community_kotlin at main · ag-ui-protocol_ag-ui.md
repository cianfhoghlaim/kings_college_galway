## README.md

A production-ready Kotlin Multiplatform client library for connecting applications to AI agents that implement the [Agent User Interaction Protocol (AG-UI)](https://docs.ag-ui.com/).

## 📚 Documentation

**[📖 Complete SDK Documentation](https://github.com/ag-ui-protocol/ag-ui/blob/main/docs/sdk/kotlin)**

The comprehensive documentation covers:

- [Getting Started](https://github.com/ag-ui-protocol/ag-ui/blob/main/docs/sdk/kotlin/overview.mdx) - Installation and quick start
- [Client APIs](https://github.com/ag-ui-protocol/ag-ui/blob/main/docs/sdk/kotlin/client) - AgUiAgent, StatefulAgUiAgent, builders
- [Core Types](https://github.com/ag-ui-protocol/ag-ui/blob/main/docs/sdk/kotlin/core) - Protocol messages, events, and types
- [Tools Framework](https://github.com/ag-ui-protocol/ag-ui/blob/main/docs/sdk/kotlin/tools) - Extensible tool execution system
```
dependencies {
    implementation("com.agui:kotlin-client:0.2.3")
}
```
```
git clone https://github.com/ag-ui-protocol/ag-ui.git
cd ag-ui/sdks/community/kotlin/library
./gradlew build
./gradlew test
```
